GenAI with PDF Docs Demo Page

Introduction

This is a simple Flask web page, which is built to link to the different stages of the demo David Spurway built to show GenAI using Sales Manual and Report PDF documents, running on Red Hat OpenShift Container Platform on IBM Power.

For this, we are using the Techzone environment "OpenShift on POWER10 - Bastion, 1 Master with NFS Storage".
